# API Lab

Минималистичное веб-приложение для тестирования API и произвольных HTTP-запросов.

## Что внутри

- **Эстетичный минималистичный интерфейс** с пресетами запросов.
- **Серверная часть на Node.js** (без внешних зависимостей).
- Встроенные API-маршруты:
  - `GET /api/health` — проверка состояния сервера.
  - `POST /api/echo` — отражение заголовков и тела запроса.
  - `POST /api/proxy` — проксирование запроса к внешнему API (с базовой защитой от локальных адресов).

## Запуск

```bash
npm start
```

После запуска открой:

- `http://localhost:3000`

## Быстрая проверка

```bash
curl http://localhost:3000/api/health
curl -X POST http://localhost:3000/api/echo -H 'Content-Type: application/json' -d '{"hello":"world"}'
```
